Reference generated from `ScreenplayKeymap.ts`, `commands.ts`, `editor-keyboard.ts`, and `ShortcutsModal.tsx`. On Windows/Linux, use **Ctrl** where macOS shows **⌘**.

## Writing

| Shortcut | Action |
|----------|--------|
| Tab | Next element / advance scene heading |
| Shift+Tab | Previous element |
| Enter | Next block / accept SmartType |
| Shift+Enter | Alternate block split |
| ⌘Z | Undo |
| ⌘⇧Z | Redo |

## SmartType (when suggestions are open)

| Shortcut | Action |
|----------|--------|
| ↑ / ↓ | Highlight suggestion |
| Enter | Accept highlighted suggestion |
| Tab | Scene-heading segment advance (on scene headings) |

## Empty block

| Shortcut | Action |
|----------|--------|
| Enter | Open element picker |
| ↑ / ↓ | Move picker highlight |
| Escape | Close picker |

Dual dialogue and inline formatting use the **floating toolbar** (mouse) — no dedicated keyboard shortcut in the current keymap.
