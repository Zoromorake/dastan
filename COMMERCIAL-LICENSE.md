# Commercial License

Dastan core is licensed under [AGPL-3.0](LICENSE). That license works for most users, self-hosters, and contributors.

Some organizations need a **proprietary license** instead:

- Embedding Dastan in a closed-source product
- Running a hosted service without publishing modifications
- Enterprise deployments requiring indemnification or dedicated support

## Contact

Commercial licensing is handled separately from the open-source repository. If you need a commercial license, contact the project maintainers with:

- Company name and use case
- Deployment model (self-hosted, SaaS, embedded)
- Expected scale (seats, documents, API volume)

## Relationship to Dastan Cloud

Paid **Dastan Cloud** subscriptions (sync, collaboration, managed AI) are product services — not a replacement for the AGPL license on the core codebase. You can use the free local app forever without a commercial license.
